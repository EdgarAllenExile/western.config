## Keybinding

| Keybind   | Action    |
|--------------- | --------------- |
| Leader + Alt + s | New Horizontal Split  |
| Leader + Alt + Shift + s | New Vertical Split  |
| Super + T / N | Create new tab / Window   |
| Super + W / Q   | Close tab / application   |
| Leader + Alt + Tab | Go Next Tab |
| Leader + Alt + Shift + Tab | Go Previous Tab |
